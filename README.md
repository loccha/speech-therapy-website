# Professional Website for Speech Therapy - [Link](https://camilleorthophonie.ca/)

Responsive website built for a speech therapy clinic, featuring a contact form with secure email handling.  
This project demonstrates responsive design, dynamic UI interactions, and secure backend practices.

## Features

- Responsive navigation with adaptive behavior based on screen size
- Mobile and tablet menu with animated transitions (fade in/out)
- Dynamic UI interactions using JavaScript and jQuery
- Client-side form validation and agreement confirmation before submission
- Server-side form handling with PHP (PHPMailer)
- Secure email configuration using environment variables (no credentials in repository)
- Input sanitization to prevent malformed or unsafe data

## Tech Stack

- HTML5
- CSS3
- JavaScript
- jQuery
- PHP
- PHPMailer (via Composer)
